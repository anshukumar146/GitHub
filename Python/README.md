# 🐍 Python Basics and Logic Practice

This folder contains fundamental Python scripts covering basic input/output, arithmetic operations, conditional logic, loops, and function-based problem-solving.

---

## 🚀 Basic Input, Output & Operators

* [01_Hello_world.py](./01_Hello_world.py) → Prints the classic "Hello, World!" message to the console.
* [02_User_input-1.py](./02_User_input-1.py) → Accepts a name via user input and prints it back.
* [03_Add_two_numbers_entered_by_user.py](./03_Add_two_numbers_entered_by_user.py) → Takes two user-inputted integers and calculates their sum.
* [04_Swap_two_numbers_(without_using_third_variable).py](./04_Swap_two_numbers_(without_using_third_variable).py) → Swaps two integers in-place using bitwise XOR operations.
* [05_Arithmetic_Operators.py](./05_Arithmetic_Operators.py) → Demonstrates all fundamental arithmetic operators in Python (addition, floor division, exponentiation, etc.).

---

## ⚖ Conditionals and Control Flow

* [06_Vote_right-1.py](./06_Vote_right-1.py) → Checks user age to determine eligibility for voting.
* [07_Check_if_a_number_is_even_or odd.py](./07_Check_if_a_number_is_even_or%20odd.py) → Checks whether a given number is even or odd.
* [08_Check_if_a_number_is_positive_negative_or_zero.py](./08_Check_if_a_number_is_positive_negative_or_zero.py) → Evaluates if a number is positive, negative, or zero using `if-elif-else`.

---

## 🔁 Loops and Iteration

* [09_Print_numbers_from_1_to_10.py](./09_Print_numbers_from_1_to_10.py) → Iterates and prints numbers from 10 down to 1 using a reverse loop.
* [10_Find_sum_of_first_N_natural_numbers.py](./10_Find_sum_of_first_N_natural_numbers.py) → Computes the cumulative sum of the first $N$ natural numbers using a `for` loop.

---

## 🧩 Functions and Mathematical Logic

* [11_Series_using_Function-1.py](./11_Series_using_Function-1.py) → Computes the sum of a factorial-based odd series ($1 + 3! + 5! + \dots + n!$) using modular functions.
* [12_Sorting_numbers_without_using_sort().py](./12_Sorting_numbers_without_using_sort().py) → Implements the Bubble Sort algorithm manually to sort a list of numbers in ascending order.
* [13_function_for_reverse_number.py](./13_function_for_reverse_number.py) → Reverses the digits of an integer using arithmetic manipulation.
* [14_check_if_a_number_is_panildrom.py](./14_check_if_a_number_is_panildrom.py) → Checks if a given number reads the same backward as forward (Palindrome check).
