# 📗 Loops and Patterns

This folder contains C programs based on loops and pattern printing. These help practice nested loops, iteration logic, and problem-solving.

---

## 📂 Programs

* [01_table_of_number.c](./01_table_of_number.c) → Prints the multiplication table of a given number.
* [02_print_N_natural_number.c](./02_print_N_natural_number.c) → Prints the first N natural numbers.
* [03_sum_of_first_N_natural_number.c](./03_sum_of_first_N_natural_number.c) → Calculates the sum of the first N natural numbers.

---

## 📐 Patterns

* [04_star_right_triangle.c](./04_star_right_triangle.c) → Prints a right-angled triangle star pattern.
* [05_reverse_right_angle_triangle.c](./05_reverse_right_angle_triangle.c) → Prints an inverted right-angled triangle star pattern.
* [06_pyramid.c](./06_pyramid.c) → Prints a pyramid star pattern.

---

## 🔢 Numerical Logic

* [07_Armstrong.c](./07_Armstrong.c) → Checks if a number is an Armstrong number.

---

gcc filename.c -o output
