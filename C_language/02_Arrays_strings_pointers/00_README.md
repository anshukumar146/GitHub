# 📘 Arrays, Strings, and Pointers

This folder contains C programs demonstrating fundamental concepts of arrays, strings, and pointers.

---

## 📊 1D and 2D Arrays

* [01_simple_array_sum.c](./01_simple_array_sum.c) → Adds two 1D arrays element-wise.
* [03_Largest_Smallest_number_in_array.c](./03_Largest_Smallest_number_in_array.c) → Finds the largest and smallest numbers in an array.
* [04_matrix_sum.c](./04_matrix_sum.c) → Calculates the sum of all elements in a 2D array.
* [05_matrix_multiplication.c](./05_matrix_multiplication.c) → Multiplies two matrices.
* [06_Array_reversal.c](./06_Array_reversal.c) → Reverses the elements of an array.
* [07_Bubble_sort.c](./07_Bubble_sort.c) → Sorts an array using the Bubble Sort algorithm.
* [14_sum_of_row_and_column_of_2D_array.c](./14_sum_of_row_and_column_of_2D_array.c) → Calculates the individual sum of each row and column in a 2D array.
* [15_principle_and_secondary_diagonal_2D_array.c](./15_principle_and_secondary_diagonal_2D_array.c) → Prints the principal and secondary diagonals of a 2D matrix.
* [16_check_for_NULL_and_Identity_matrix.c](./16_check_for_NULL_and_Identity_matrix.c) → Checks if a matrix is a NULL or Identity matrix.
* [17_Return_of_even_and_odd_element_in_1D_array.c](./17_Return_of_even_and_odd_element_in_1D_array.c) → Returns the respective sums of all even and odd elements in an array.

---

## 🔤 Strings and Text Processing

* [08_String_length.c](./08_String_length.c) → Finds the length of a string without using `<string.h>`.
* [09_String_reverse.c](./09_String_reverse.c) → Reverses a string in-place.
* [10_String_Palindrome.c](./10_String_Palindrome.c) → Checks if a string is a palindrome.
* [11_Japanese_Vocab_search.c](./11_Japanese_Vocab_search.c) → Searches for vocabulary from Minna no Nihongo Lessons 1 & 2.
  * *Example words:* watashi, anata, sensei, kyoushi, gakusei, kaishain, shain, ginkouin, isha, kenkyuusha, enjinia, daigaku, byouin, denki
* [13_Anagram.c](./13_Anagram.c) → Checks if two strings are anagrams of each other.
* [18_replace_all_the_alphabet_with_@_sign_digits_with_#_and_special_characters_with_$.c](./18_replace_all_the_alphabet_with_@_sign_digits_with_#_and_special_characters_with_$.c) → Replaces alphabets with `@`, digits with `#`, and special characters with `$`.

---

## 📍 Pointers

* [02_array_addition_pointer.c](./02_array_addition_pointer.c) → Adds two arrays using pointers.
* [12_Swap_number_using_pointer.c](./12_Swap_number_using_pointer.c) → Swaps two numbers using pointers and a temporary variable.
