# 💾 DMA, Structures, and File Handling

This folder contains C programs based on Dynamic Memory Allocation (DMA), structures, and file I/O operations to practice clean data management and memory control.

---

## 🏢 Structures and User-Defined Types

* [01_percentage_calculation_when_three_marks_given.c](./01_percentage_calculation_when_three_marks_given.c) → Calculates the total percentage of a student based on three subject marks stored in a structure.
* [02_Add_n_days_to_a_date.c](./02_Add_n_days_to_a_date.c) → Adds a specific number of days to a given date structure while handling variable month lengths and leap years.
* [05_Structure_with_Function.c](./05_Structure_with_Function.c) → Manages an array of student record structures to track and print details along with their percentage.

---

## 📂 File Handling

* [03_BASIC_FILE_HANDLING.c](./03_BASIC_FILE_HANDLING.c) → Reads characters from user input into a master file, then filters and sorts them into separate text files based on whether they are alphabets, numbers, or special characters.

---

## 🧠 Dynamic Memory Allocation (DMA)

* [04_Number_larger_smaller_or_equall_then_X.c](./04_Number_larger_smaller_or_equall_then_X.c) → Dynamically allocates an array using `calloc()`, prompts for a target value, and counts how many array elements are larger than, smaller than, or equal to that value.
