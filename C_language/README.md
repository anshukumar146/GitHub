## 📂 Repository Structure

Each folder targets a specific proficiency level or module in C:

* 📁 [01_Basic_c_programs](./01_Basic_c_programs) → Entry-level code covering fundamental input/output, conditional structures, basic operators, and basic logic.
* 📁 [02_Arrays_strings_pointers](./02_Arrays_strings_pointers) → Programs focusing on sequential data structures, matrix operations, string manipulations, and pointer arithmetic.
* 📁 [03_Loops_and_pattern](./03_Loops_and_pattern) → Logic-heavy practice handling iterative loops, multi-layered nested loops, and console pattern-printing shapes.
* 📁 [04_DMA_STRUCTURE_FILE_HANDLING](./04_DMA_STRUCTURE_FILE_HANDLING) → Advanced concepts including dynamic allocation blocks (`malloc`/`calloc`), user-defined structures, and permanent file storage handling.
